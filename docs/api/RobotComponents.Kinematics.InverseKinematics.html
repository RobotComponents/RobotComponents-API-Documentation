CalculateClosestRobotJointPosition(RobotJointPosition)

Calculates and returns the closest Robot Joint Position to a given previous Robot Joint Position. This methods sets and returns the closest Robot Joint Poistion insides this Inverse Kinematics object. You first have to calculate the Inverse Kinematics solution before you call this method. This method is typically used for using the Auto Axis Config inside the Path Generator.
