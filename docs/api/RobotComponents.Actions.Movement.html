Properties

DigitalOutput

Gets or set the Digital Output. If an empty or invalid Digital Output is set a normal movement will be set (MoveAbsJ, MoveL or MoveJ). If a valid Digital oOutput is combined movement will be created (MoveLDO or MoveJDO). If as Movement Type an MoveAbsJ is set an extra RAPID code line will be added that sets the Digital Output (SetDO).
