UserFrame

Gets or sets the user coordinate system, i.e. the position of the current work surface or fixture. If the robot is holding the tool, the user coordinate system is defined in the world coordinate system (in the wrist coordinate system if a stationary tool is used). For movable user frame (FixedFrame = false), the user frame is continuously defined by the system.
